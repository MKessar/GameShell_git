Mission goal
============

A dragon is approaching, you have to stop it or it will burn the castle and the whole region. You need an enchantment able to kill dragons.
Wizards of Al Jeit castle were the first ones to work on such an enchantment.

Go to the portal room and clone the portal of castle Al Jeit. We will figure out if it can be of any help.
All adresses of castle portals are registered in the portal book in the library somewhere in the main tower.


Useful commands
===============

To clone a repository
git clone REPOSITORY_URL
