Mission goal
============

Most of the next missions need git to be configured in English.

This is necesseary to avoid bugs with the current version of this game.


Details
-------

To avoid typing this long command each time, you can create an alias 
just like

   alias git='LANG=en_GB git'


Useful commands
===============



alias STRING='COMMAND'
  Create a synonym for a string, that will stand for a command.
