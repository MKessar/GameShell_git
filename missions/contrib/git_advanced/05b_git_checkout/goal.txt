Mission goal
============


You are familiar with the history of modifications.

Now, you need to display the most recent version of your spellbook.



Useful commands
===============

git log --pretty=oneline : will display the list of previous commit


git checkout branchname: will checkout the most recent commit of the branch "branchname"

git branch: display your current branch and the list of available branches.
